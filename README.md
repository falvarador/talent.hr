# 🏢 HR Management System - Test Project

Este es un proyecto de prueba para un sistema de gestión de Recursos Humanos (HR). La aplicación permite administrar empleados, departamentos y roles, integrando un backend robusto en Java con un frontend dinámico en ReactJS.

## 🚀 Tecnologías Utilizadas

### Backend
* **Java 17 / 21**
* **Spring Boot 4.x** (Spring Data JPA, Spring Security)
* **H2 Database / MySQL** (Configurable)
* **Maven** (Gestor de dependencias)

### Frontend
* **ReactJS 18+**
* **Vite** (Herramienta de construcción)
* **Tailwind CSS** (Estilos)
* **Axios** (Cliente HTTP)
* **React Query / Context API** (Gestión de estado)
